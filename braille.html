SONIC BRAILLE

Encoding
Each character is converted to a 6-dot Braille cell. The cell's binary value (0–63) maps to a pitch in your chosen scale.
Latin
International Braille — Grade 1, one cell per letter.
日本語
Japanese Braille (点字) — kana to Braille cells. Dakuten/handakuten as prefix cells.
한국어
Korean Braille (한국 점자) — jamo decomposition, each jamo as a cell.
中文
Chinese Braille (现行盲文) — pinyin-based, initial + final as cells, no tones.
Pitch Mapping
binary value % scale_size → degree, ⌊value / scale_size⌋ → octave offset, clamped to range. Toggle { } to see full derivation.
